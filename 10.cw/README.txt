How to make this software work? You will need to populate an .ini file with the sftp values for your site. One can use 2.initialize.pl to create the .ini file, or just take 5.example.ini, put your values in RHS and place it somewhere you think is safe on your computer. Then put this path as the value for ini_path in the sub init_values of located in html8.pm . Also, other values in the main data hash will need to be cutomized. Then, open up the base directory, chmod + x 10.cw1.pl, and then run it.
